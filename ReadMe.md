# 🚆 NeuroRail – Smart Railway Surveillance

AI-powered system to detect unattended baggage using YOLOv8.

## 🔍 Features
- Person & bag detection
- Real-time tracking

## 🧠 Model
- YOLOv8 trained on custom dataset
- Classes: person, bag, obstacle
